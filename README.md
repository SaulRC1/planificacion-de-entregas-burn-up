SOBRE EL PROGRAMA
-------------------------------------------------------
Práctica final para CMEPPS, asignatura de 4º curso de Ingeniería Informática, Universidad de Huelva.

El programa consiste en la creación y planificación de proyectos mediante la metodología burn-up.
